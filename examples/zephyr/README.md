# Examples: Zephyr

The examples in this directory demonstrate how to do basic GPIO and serial
operations in Zephyr.

Key Tasks:
- Implement GPIO, UART, and USB CDC drivers for Zephyr
- Learn how to get Zephyr's build system to build drivers structured with C
  shims calling Rust drivers through an FFI wrapper
- Learn how to get Zephyr's build system to sign binaries for Bao1x bootloader
- Learn how to get Zephyr's build system to make UF2 files
